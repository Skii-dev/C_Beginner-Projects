## üöÄ How to Run My Projects

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Skii-dev/C_Beginner-Projects.git](https://github.com/Skii-dev/C_Beginner-Projects.git)
    ```
2.  **Open in CLion:**
    * Open CLion and select "Open Project."
    * Choose the `C_Beginner-Projects` root folder.
    * CLion will automatically load the `CMakeLists.txt` files for each sub-project.
3.  **Select & Run:**
    * Use the **Run Configuration** dropdown (top right) to select the specific project you want to test.
    * Press **Shift + F10** to compile and run.
